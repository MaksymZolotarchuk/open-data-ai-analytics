# open-data-ai-analytics

## Мета
Проаналізувати дані про транспортні засоби, зареєстровані у 2024 році, з метою виявлення:
- структури автопарку (типи ТЗ, паливо, марки)
- регіональних особливостей реєстрацій
- вікової структури транспортних засобів
- тенденцій оновлення автопарку

## Джерело відкритих даних
Портал відкритих даних України: https://data.gov.ua/

Датасет: Дані щодо транспортних засобів за 2024 рік

## Питання / гіпотези для аналізу
1. У 2024 році переважають легкові автомобілі з дизельним або бензиновим типом палива.
2. Більшість зареєстрованих транспортних засобів — це вживані авто (MAKE_YEAR < 2024).
3. Існує залежність між типом транспортного засобу (KIND) та типом палива (FUEL).
4. У великих регіонах (REG_ADDR_KOATUU) кількість реєстрацій суттєво вища.
5. Нові транспортні засоби (N_REG_NEW) частіше належать юридичним особам (PERSON = J).
