# open-data-ai-analytics

## Мета
Проєкт демонструє повний пайплайн роботи з відкритими даними: завантаження, перевірка якості, дослідження/моделювання та візуалізація.

## Джерело відкритих даних
Портал відкритих даних України: https://data.gov.ua/

> У проєкті використовується обраний датасет з data.gov.ua (посилання на конкретний набір даних додайте після вибору).

## Питання / гіпотези для аналізу
1. Які ключові тренди/зміни спостерігаються у даних у часі (якщо є дата/період)?
2. Чи є суттєві відмінності між групами (регіони/категорії/типи) за основними показниками?
3. Чи можна побудувати модель (класифікацію/регресію/кластеризацію), яка дає корисні інсайти (наприклад, прогноз або сегментацію)?

## Структура репозиторію
- `src/data_load` — завантаження даних
- `src/data_quality_analysis` — перевірка якості
- `src/data_research` — аналіз/моделі
- `src/visualization` — графіки/дашборди
- `notebooks/` — EDA/експерименти
- `reports/figures/` — збережені графіки
